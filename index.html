Portfolio

  • Memory Cards (2022)

    Memory Cards is a flashcard-creator website. Users can create sets to store their flashcards. Each flashcard can be flagged either green, amber or red to record your proficiency/understanding of that flashcard. You can also order your sets based on how many flashcards are in it or by how many have been flagged red or green.

    It has been built using Flask, a Python web framework. For the front-end of the website, I have used HTML/CSS with Bootstrap.

    Memory cards features a PostgreSQL database using the sha-256 hashing algorithm to hash and store sensitive data.

    For the different functions of the site, I have made several API calls to a database to fetch, store, modify and delete data.

  • Finerfocus:
    Focus and Study Timer (2021)

    Finerfocus is an Android app designed to improve focus and help those who are studying or working. Users can create custom length timers, get reminders at particular points during their timer, and have a wide range of white noises play in the background of their timer to aid focus.

    The app was developed using the Flutter software development kit in the dart programming language, which was new to me when developing the app.

    This project taught me what is involving in deploying advertisements and managing app revenue using Google Admob.
